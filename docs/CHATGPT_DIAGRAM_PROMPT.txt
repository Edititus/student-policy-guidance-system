Create a professional, visually appealing diagram showing the RAG (Retrieval-Augmented Generation) architecture for an AI-based student policy guidance system. The diagram should be suitable for a PowerPoint presentation or thesis document.

**System Name:** Veritas University Policy Guidance System

**Diagram Requirements:**
- Clean, modern design with a light or professional color scheme
- Single-page overview that fits on one slide
- Show the complete flow from policy upload to student receiving answers
- Include icons/symbols where appropriate
- Label all major components clearly

**Components to Include:**

1. **INDEXING PHASE (Left/Top):**
   - Admin uploads policy documents (PDF, DOCX, TXT)
   - Document parsing and text extraction
   - Text chunking (1000 characters with 200 character overlap)
   - Embedding generation using OpenAI text-embedding-3-small
   - Storage in Vector Database (show as cylinder/database icon)

2. **RETRIEVAL PHASE (Middle):**
   - Student asks a question in natural language
   - Query embedding (convert question to vector)
   - Semantic search using cosine similarity
   - Retrieve top 5 most relevant policy chunks (similarity > 0.7)
   - Show search happening in the Vector Database

3. **GENERATION PHASE (Right/Bottom):**
   - Augment query with retrieved context
   - Send to GPT-4o-mini for answer generation
   - Confidence scoring (HIGH ≥0.85, MEDIUM 0.70-0.85, LOW <0.70)
   - Return answer with policy sources cited
   - Display to student with confidence badge

**Technical Details to Show:**
- Embedding Model: text-embedding-3-small (1536 dimensions)
- Generation Model: GPT-4o-mini
- Response Time: < 3 seconds
- Vector Database with semantic search capability
- Cosine similarity algorithm for matching

**Visual Style:**
- Use arrows to show data flow direction
- Use different colors for each phase (e.g., blue for indexing, green for retrieval, orange for generation)
- Include small icons: 📄 for documents, 🤖 for AI models, 💾 for database, 👨‍🎓 for student
- Professional and clean - suitable for academic presentation
- Modern infographic style

**Layout Suggestion:**
Either a left-to-right horizontal flow or a circular/cycle flow showing how policies are indexed once and then queried many times.

**Output Format:**
Create a high-resolution diagram (at least 1920x1080) that can be inserted into PowerPoint or a thesis document.
